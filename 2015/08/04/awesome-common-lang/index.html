commons-lang这货是让我没法造轮子啊

  • Java
  • commons-lang
  • 工具包

自从参加工作以后，一直有一个想法就是想写一些自己的基础类库，工具类吧啦吧啦之类的，但是越到现在越发现，这个想法是多余了的。越来越觉得Java的强大之处在于你想要的东西或者想造的轮子，在某一天，突然得知原来早就有了现成的十分健全的库可供使用，不得不感叹Java的大而全，也发现自己的积累不足。

这不，commons-lang就是一个，平时用的比较多的API，mark之。

简介

引用官网Apache Commons Lang

Lang provides a host of helper utilities for the java.lang API, notably String manipulation methods, basic numerical methods, object reflection, concurrency, creation and serialization and System properties. Additionally it contains basic enhancements to java.util.Date and a series of utilities dedicated to help with building methods, such as hashCode, toString and equals.

API文档URL为commons-lang/javadocs/api-3.4，目前最新版本为3.4，需要在maven中引入以下配置

其中，记录一些最常用的类吧。

StringUtils

这个工具类真是让我有一种相见恨晚的感觉，自从用上后，再也不用担心字符串的空指针异常啦。几乎封装了所有的String类的API，只是把String作为参数传值处理。
